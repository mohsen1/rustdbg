rdbg

IDE-grade Rust debugging for coding agents. Set breakpoints, run, and read the real runtime values of variables — instead of guessing from println! and recompiling.

# macOS and Linux
curl -fsSL https://azimi.me/rust-debugger-skill/install.sh | sh

It also needs rust-analyzer (rustup component add rust-analyzer) and lldb-dap.

What it does

  • Breakpoints: line, function, conditional, hit-count, logpoint, panic, and watchpoints.
  • Step over / in / out, run-to-line, pause, restart.
  • Read locals as real Rust values, evaluate paths, and change a variable live.
  • Jump to definition, hover, and find references via rust-analyzer.

Use it

As a skill (the rdbg CLI) or as an MCP server (rdbg mcp) in Claude Code and Codex. See the README.
